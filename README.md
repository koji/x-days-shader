# X Days Shader

## how to run

### install dependencies

```shell
cd x-days-shader


# bun
bun install

# pnpm
pnpm install

# npm
npm install
```

### run dev server

```shell
# bun
bun run dev

# pnpm
pnpm run dev

# npm
npm run dev
```

### build the app

```shell
# bun
bun run build

# pnpm
pnpm run build

# npm
npm run build
```
